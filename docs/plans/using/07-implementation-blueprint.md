- SWITCH pattern 在 subject 可推断为 union 时支持 Rust-like 未限定 variant case：`(Variant)` / `(Variant(x))` / `(Variant { field: local })` 会按 subject 类型补全 union，参与 tag 比较、payload 绑定和穷尽性检查；重复覆盖同一 variant 的第二个 case 会作为不可达 case 报错。这条补全只用于 identifier / primary-expression case，裸 `[x]` / `{field}` 默认展开仍限定在 `using` 默认 `@` variant 语义里。owner payload 默认借用为 `Borrow<T>`；显式 `move` 可写作 `(Variant(move local))`、`(Variant { field: move local })` 或同名 `(Variant { move field })`，并复用 `SET_MEMBER_SLOT_NULL` 清空 matched payload。
