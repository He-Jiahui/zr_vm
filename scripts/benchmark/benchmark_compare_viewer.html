全量：各 case 平均耗时 (ms) — 相对本行最短值着色

解释器 / 脚本虚拟机类 — 同类互比

仅包含：Python、Node.js、Lua、QuickJS、ZR interp。着色与条形比例均只在本组实现之间比较（不与 C / AOT 等混排）。

仅包含：C、Rust、Java、.NET、ZR binary、ZR aot_c、ZR aot_llvm。Java / .NET 为虚拟机内 JIT 编译到机器码的执行路径。

全量：按 case 条形对比（含所有实现）

每个 case 内按耗时升序；若需只看解释器组或只看机器码组，请使用上方分组表。

数值来源：benchmark_report.json 中 summary.mean_wall_ms；SKIP/FAIL 显示为 “—”。全量表与各分组表使用同一数据源，分组列由实现名称归类（见各节说明）。
